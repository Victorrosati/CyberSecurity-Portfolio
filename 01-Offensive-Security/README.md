# Offensive Security Reports
# 01 - Offensive Security & Red Team
This folder contains reports focused on identifying and exploiting vulnerabilities in web applications and systems.

### Key Projects:
* **BlackBox Series:** Complete penetration tests on isolated machines (Jangow, Lupin, Harry Potter).
* **Web Exploits:** Manual SQL Injection via Burp Suite and Stored XSS for Session Hijacking.
* **System Exploits:** Analysis of Buffer Overflow and exploitation of services like Samba and Tomcat.
